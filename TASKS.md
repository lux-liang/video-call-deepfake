# TASKS

## P0 / 最小可演示闭环

- 上传会议视频接口占位存在，支持后续接入文件存储与任务创建。
- 返回 mock/sample 结构化分析结果。
- 展示参会者风险卡片所需稳定字段。
- 展示异常事件列表。
- 展示可疑时间段。
- 展示工具调用日志。
- 提供自动巡检报告的 sample 输出。
- README 可读，能解释系统目标、架构、限制与启动方式。
- demo 流程可录，样例数据与脚本存在。

## P1 / 真实 pipeline 接入与 UI 打磨

- 接入 ffmpeg 做视频切片、抽帧、音轨拆分。
- 接入 VAD、OpenCV、MediaPipe 等外部工具做基础特征抽取。
- 建立 ingestion -> parsing -> event extraction -> scoring 的最小真实链路。
- 增加任务轮询与结果详情页面。
- 打磨前端风险卡片、时间线、证据解释和报告导出。
- 增加 schema 校验、字段完整性检查与失败重试。
- 接入报告生成与结构化导出。

## P2 / 增强能力

- 接入更接近实时流的分析链路。
- 支持多视频批处理和任务队列调度。
- 增加多会议对比分析与风险趋势。
- 引入更细粒度的行为特征与多模态证据融合。
- 增强人机复核、标注反馈和主动学习机制。
- 增强可观测性、成本控制和权限管理。
